Three Layers

Component Actors

Each K8s component is an independent XState actor. Owns its state machine, generates messages on transitions, responds to incoming events. No knowledge of other actors.

Event Bus

Decouples actors. Records event log, buffers events, delivers on PlaybackController signal. Emits state change notifications to UI. Manages snapshots.

Playback Controller

Controls simulation flow. Tells Event Bus when to deliver the next event. Manages auto-play speed, snapshots for backward navigation. State: idle/running/paused/completed.

Key Invariants

  • Actors never talk to each other directly вҖ” all communication through Event Bus
  • Actors never talk to React вҖ” Event Bus emits state changes to Zustand
  • Playback Controller never touches Actor internals вҖ” it only controls event delivery timing
  • Each Actor can be tested in isolation by feeding events and observing outputs
